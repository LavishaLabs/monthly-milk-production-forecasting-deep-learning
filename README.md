# Monthly Milk Production Forecasting using RNN, LSTM, and GRU
## Project Overview
This project applies Deep Learning Time Series Forecasting techniques to predict
future monthly milk production.
Three neural network architectures were implemented:
- RNN
- LSTM
- GRU
---
## Business Objective
The forecasting system helps dairy businesses:
- Optimize inventory
- Improve supply chain planning
- Manage seasonal production
- Reduce operational inefficiencies
---
## Dataset Information
- Monthly milk production data
- Time series dataset
- Historical production observations
---
## Technologies Used
- Python
- Pandas
- NumPy
- TensorFlow
- Keras
- Matplotlib
- Scikit-learn
---
## Project Workflow
1. Exploratory Data Analysis
2. Data Scaling
3. Sequence Generation
4. RNN Model
5. LSTM Model
6. GRU Model
7. Forecasting
8. Model Evaluation
9. Future Prediction
---
## Evaluation Metrics
- RMSE
- - MAE
- MAPE
---
## Results
LSTM and GRU models achieved superior forecasting performance compared to the
basic RNN model.
